ИНСТРУКЦИЯ
//***********************************************************************************************************//
РАБОЧИЙ РЕЖИМ-СВАРКА:

АКТИВАЦИЯ: Нажатие кнопки, подключенной из пина START на пин GROUND (к минусу питания) или Энкодер по часовой стрелке
**на пин FIRE_SIGNAL подаются сварочные импульсы (сюда подключаем реле управления сварочным током)

ДЕАКТИВАЦИЯ: Автоматическая или Энкодер против часовой стрелки

РЕЖИМ НАСТРОЙКИ ВРЕМЕНИ РЕЖИМОВ СВАРКИ:
АКТИВАЦИЯ: Нажатие кнопки, подключенной из пина SW на пин VCC (к плюсу питания)

!!! НЕ ЗАБЫВАЙТЕ, Arduino Nano РАБОТАЕТ ОТ 2,7 до 5,5в НАПРЯЖЕНИЯ. ТАКОГО ЖЕ НОМИНАЛА СИГНАЛ БУДЕТ И НА ВЫХОДЕ ПИНА FIRE_SIGNAL!!!
!!УЧИТЫВАЙТЕ ЭТО ПРИ ВЫБОРЕ УПРАВЛЯЮЩЕГО РЕЛЕ!!

//***********************************************************************************************************//


=> T1: ПЕРВЫЙ ИМПУЛЬС (РАЗОГРЕВАЮЩИЙ)        P: ПАУЗА МЕЖДУ ИМПУЛЬСАМИ		
     T2: ВТОРОЙ ИМПУЛЬС (СВАРКА)/(Выход)                          <Информация>


	ЭНКОДЕР - смена пункта меню
	SW - Вход в режим редактирования 
	
Режим редактирования: (** На экране в правом нижнем углу "<#>")

=>ПЕРВЫЙ ИМПУЛЬС, ПАУЗА МЕЖДУ ИМПУЛЬСАМИ,
	SW - выход из режима редактирования данного тайминга
	ЭНКОДЕР - смена времени тайминга
	
=>ВТОРОЙ ИМПУЛЬС (T2 -СВАРКА)
	SW - выход из режима настроек + сохранение ваших настроек в память EEPROM
	ЭНКОДЕР - смена времени тайминга
//***********************************************************************************************************//

P.S.
Переменная #define TIME_PRE_MS 100 устанавливает паузу перед сварочным импульсом. 
Если она вам не нужна - поставьте 0 вместо 100, не удаляйте переменную совсем!
